PUSHF/PUSHFD - Push Flags onto Stack





	Usage:	PUSHF
		PUSHFD	(386+ only)
	Modifies flags: None


	Transfers the Flags Register onto the stack.  PUSHF saves a 16 bit
	value while PUSHFD saves a 32 bit value.


				 Clocks			Size
	Operands	 808x  286   386   486		Bytes

	none		10/14	3     4	    4		  1
	none  (PM)	  -	-     4	    3		  1
