Q 1 - Choose the command to print the current working directory

A - cwd

B - pwd

C - wd

D - nwd

Answer : B

Explanation

pwd (present working directory).

Q 2 - How does a comment can begin in a shell script file.

A - Beginning with #

B - Beginning with $

C - Beginning with !

D - None of the above.

Answer : A

Explanation

A comment line begin with # (pound symbol) and is ignored by the shell.

Q 3 - Which command can be used to delete a file?

C - rm

D - delete

Answer : C

Explanation

rm, can delete a file as well as a directory.

Explanation

–r mean recursively and –f means forcibly.
